# shell-switcher
I created this shell switcher to work with fish and bash. I like both on my cachyos machine and was not ready to give one for another, so its a very simple script to switch between shell type in linux.

steps to use:

1)Elevate the access permissions of  "shell_switcher.sh" by typing chmod +x (this gives the script executable permission).

2)Run the script by either sh ./shell_switcher.sh or just ./shell_switcher.sh.
